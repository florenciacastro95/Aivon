# AIVON - Parcial Ingeniería de Software

Implementación en Java del ejercicio propuesto para la empresa de productos cosméticos **AIVON**.

## Descripción

El proyecto modela una empresa de venta por catálogo, donde las revendedoras realizan pedidos durante campañas, gestionan cuentas y pueden transferir dinero entre sí.
Implementé las pruebas unitarias solicitadas en el enunciado utilizando JUnit.

## Tecnologías utilizadas
- Java
- JUnit 4
- IntelliJ IDEA
- Maven

## Estructura del proyecto

```
src
├── aivon.modelo
│   ├── Caja.java
│   ├── Campania.java

│   ├── Revendedora.java
│   ├── Transportista.java
│   └── Vehiculo.java
│
└── aivon.test
│   ├── RevendedoraTest.java
│   └── ParameterRevendedoraTest.java
└── main.java   
```

## Pruebas

Para probar las clases deberás ejecutar el archivo main que ejecuta lo que debieron ser los diagramas de secuencias que fueron quitados del parcial. Aparecerán las pruebas por consola.
Los archivos de testing deberán ser ejecutados directamente.
